# Python Exercises — Buffet Business Context

Practice exercises using real business data from an events buffet, built as part of a self-directed learning path toward a junior data/backend role.

Built without AI autocomplete or code suggestions — every exercise was written by hand to develop genuine understanding of core logic (loops, conditionals, data structures) before moving to higher-level tools like pandas.

## Exercises

- **ex01** — Event classification by guest count (list comprehension, dictionaries)
- **ex02** — Available date filtering and sorting (dictionary iteration, sorted())
- **ex03** — Revenue aggregation by service category (manual dictionary accumulation — no pandas)
- **ex04** — Introduction to pandas: reading CSV, filtering rows, groupby aggregation
- **ex05** — Applied pandas: grouping buffet order data by type and item, mirroring a real weekly procurement process

## Note on data

All data in these files is fictional or anonymized. No real client information is included.
